# Guess the Output C++

A fun, interactive C++ learning game where players predict the exact output of code snippets or identify compilation/runtime errors. Perfect for reinforcing C++ concepts through challenging "guess the output" problems.

## How to Play

1. **Read the code** - Study the provided C++ program carefully
2. **Guess the output** - Enter what you think will be printed to stdout, or select an error type
3. **Get instant feedback** - See if you're right and learn why with detailed explanations

## Features

- **Progressive difficulty** (1-5 levels)
- **Multiple problem types**: stdout output, compilation errors, runtime errors, undefined behavior
- **Educational explanations** for every problem
- **C++ concept-focused** problems covering pointers, memory, STL, templates, and more
- **Single-file programs** for easy understanding

## Tech Stack

- **Frontend**: Next.js
- **Backend**: None (static site)
- **C++ Version**: C++17
